# RV08: ASM_recursion

V zbirnem jeziku napišite rekurzivno funkcijo, ki izračuna n-ti člen zaporedja:

f(n) = 1 + f(n-3) + f(n-2), n > 2, f(0) = 12, f(1)=22, f(2) = 45


V glavnem programu izračunajte člene od 3 do 10 in rezultat izpišite.
